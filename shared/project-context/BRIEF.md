# Project Brief — Pixel Art Resume

## Vision
An interactive resume website where visitors control a pixel art character that walks through a side-scrolling world (inspired by Super Mario). As the character encounters different locations and objects, the corresponding resume information is displayed below the game canvas.

## Target Audience
- Recruiters and hiring managers
- Potential clients or collaborators
- Tech community / portfolio viewers

## User Journey
1. **Landing**: Page loads with pixel art world and character
2. **Onboarding**: Brief instruction — "Press arrow keys or tap to walk"
3. **Exploration**: User walks right through a themed landscape
4. **Education Zone**: Pass a school building → Education section appears below
5. **Experience Zone**: Walk past office buildings → Work experience section appears
6. **Skills Zone**: Enter a tech lab area → Skills section with visualizations
7. **Projects Zone**: Visit a workshop → Project cards appear
8. **Contact Zone**: Reach the finish → Contact info and call-to-action

## Core Mechanics
- **Movement**: Left/Right arrow keys, spacebar/up to jump
- **Mobile**: On-screen touch controls (d-pad + jump button)
- **Interaction**: Walk into trigger zones to reveal resume content
- **Camera**: Follows player, smooth horizontal scrolling

## Non-Goals (v1)
- No combat or enemies
- No score system
- No user accounts or backend
- No multiplayer
- Keep it simple — it's a resume, not a full game

## Success Metrics
- Loads in < 3 seconds
- 60fps on mid-range devices
- All resume content accessible via keyboard/screen reader
- Works on Chrome, Firefox, Safari, Edge + mobile
- Bundle size < 2MB
